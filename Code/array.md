# array

```cpp
#include <array>

array::at 指向元素
array::back 指向最后一个元素
array::begin 指向第一个元素的地址
array::cbegin
array::cend
array::crbegin
array::crend
array::data 返回一个指向数组中的第一元素对象。
array::empty 检查一个数组是否为空
array::end 指向最后一个元素的地址
array::fill 对数组进行填充
array::front 指向第一个元素
array::max_size
array::operator[]
array::rbegin 返回一个指向反向容器的第一个元素的定位器。
array::rend 返回一个指向反向容器的最后一个元素的定位器。
array::size
array::swap 交换两个数组的元素
```
